🛒 Add to Cart - React
A simple React-based shopping cart application that allows users to add items to a cart, view the cart size, and get alerts when an item is already added.

🚀 Features
✅ Display products from a list
✅ Add items to the cart
✅ Show cart size in the navbar
✅ Prevent duplicate items in the cart
✅ Simple and responsive UI

🛠️ Tech Stack
React.js - Component-based UI

useState Hook - Manage cart state

CSS - Styled components

FontAwesome - Icons for UI

📂 Project Structure
/src
 ├── /components
 │   ├── Navbar.js       # Displays the cart size
 │   ├── Shop.js         # Renders the product list
 │   ├── Card.js         # Individual product card
 ├── /styles
 │   ├── navbar.css      # Navbar styles
 │   ├── card.css        # Product card styles
 ├── list.js             # Product data
 ├── App.js              # Main component
 ├── index.js            # Entry point
 ├── README.md           # Documentation
